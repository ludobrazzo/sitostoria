# Sito di storia interattivo

Creato dal file PDF nello ZIP caricato.

## Come usarlo
1. Estrai lo ZIP.
2. Apri `index.html` nel browser.
3. Entra negli argomenti in ordine.
4. Alla fine di ogni pagina completa il quiz: se anche una risposta è sbagliata o non selezionata, il quiz si resetta da capo.

## Struttura
- `index.html`: indice generale.
- `argomenti/`: una pagina HTML per ogni argomento.
- `assets/pages/`: immagini delle pagine originali del PDF.
- `style.css`, `topics.js`, `app.js`: grafica, dati e logica dei quiz.
